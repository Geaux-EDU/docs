title: "Comparison with the Heroku CLI"
---

Following is the one-to-one comparison of some of the commands supported by the Heroku CLI and their nearest Porter counterparts.

| Heroku                | Porter                                                                                  |
| --------------------- | --------------------------------------------------------------------------------------- |
| heroku addons         | [porter list addons](/enterprise/cli/command-reference/porter-list#porter-list-addons)             |
| heroku addons:create  | [porter create](/enterprise/cli/command-reference/porter-create)                                   |
| heroku addons:destroy | [porter delete addon](/enterprise/cli/command-reference/porter-delete#porter-delete-addon)         |
| heroku apps           | [porter list applications](/enterprise/cli/command-reference/porter-list#porter-list-applications) |
| heroku apps:create    | [porter create](/enterprise/cli/command-reference/porter-create)                                   |
| heroku apps:destroy   | [porter delete app](/enterprise/cli/command-reference/porter-delete)                               |
| heroku auth:login     | [porter auth login](/enterprise/cli/command-reference/porter-auth)                                 |
| heroku auth:logout    | [porter auth logout](/enterprise/cli/command-reference/porter-auth)                                |
| heroku auth:token     | [porter config](/enterprise/cli/command-reference/porter-config)                                   |
| heroku config:set     | [porter update env-group set](/enterprise/cli/command-reference/porter-update)                     |
| heroku config:set     | [porter update env-group unset](/enterprise/cli/command-reference/porter-update)                   |
| heroku logs           | [porter logs](/enterprise/cli/command-reference/porter-logs)                                       |
| heroku ps:forward     | [porter kubectl -- port-forward](/enterprise/cli/command-reference/porter-kubectl)                 |
| heroku run            | [porter run -e](/enterprise/cli/command-reference/porter-run)                                      |
